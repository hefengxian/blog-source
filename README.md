# 博客源码

基于Hexo <s>Apollo</s> NexT主题

# 安装使用

安装Hexo，可以参照[Hexo](https://hexo.io/)官网的文档

本地测试
```bash
$ npm install --registry=https://registry.npm.taobao.org

$ hexo server
```

部署
```bash
$ hexo generate --deploy
```
